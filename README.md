Plik zawiera kod do prostej strony internetowej sklepu utworzonej przy wykorzystaniu języka HTML. Znajduje się na niej krótki opis sklepu i jego oferta, 
odnośniki do innych stron np. wp.pl, przykładowy formularz do wypełnienia oraz filmik dotyczący oferty sklepu.
